# Структура репозитория
- project/ - текущая версия проекта
  - test_vectors/ - txt файлы для теста модулей
  - tb/ - модули тестирования
  - scripts/ - скрипты для работы с проектом
  - rtl/ - RTL-описание разрабатываемого фильтра
  - constraints - констрейны для тестирования проекта на ПЛИС
- gold_models/ - эталонные модели
  - name/ - папка эталонной модели
    - README.md - описание эталонной модели
- materials/ - статьи и прочие учебные материалы
  - name/ - папка соответствующего материала
    - README.md - поясняющая информация к материалу
- reports/ - отчёты для Ядра
  - report_n/ - n-ый по счёту отчёт для Ядра
    - report - отчёт для Ядра
    - task - задание на текущий созвон
    - pictures/ - иллюстрации к отчёту (svg, png и прочее)
    - scripts/ - скрипты использованные для формирования отчёта
- README.md - файл с описанием проекта и актуальными заданиями
- TASKS.md - файл с заданиями по проекту
