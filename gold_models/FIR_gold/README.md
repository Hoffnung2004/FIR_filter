**createScientificPlotC** - функция для отрисовки графиков. Функционал описан внутри файла функции
**getGradient**           - функция для создания массива цветов для графика. Функционал описан внутри файла функции
**getGradientColor**      - функция для получения конкретного цвета для графика. Функционал описан внутри файла функции
**quantFilterIntOutWL** - модель КИХ фильтра для условия, что при промежуточных вычислениях достаточная разрядность для хранения результатов. В случае если разрядности не хватает, может выступать в качестве эталонной модели для фильтра с последовательным набором суммы. Все числа трактуются как целые. Таким образом перед использованием функции следует привести к достаточной величине.
	signal_raw - входной сигнал фильтра (double)
	taps - импульсная характеристика фильтра (double)
	Wsignal - битность входного сигнала
	Wtaps - битность коэфициентов
	Wout - битность, используемая для вычислений
	Wfinal - битность, к которой следует привести финальный результат
	roundMethod - метод округления
**main** - основной скрипт - формирует различные тестовые воздействия
